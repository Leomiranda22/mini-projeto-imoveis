
# Mini Projeto: Análise de Imóveis com Pandas

## Sobre o projeto

Este projeto apresenta uma análise exploratória de uma base de dados de imóveis para aluguel utilizando Python e a biblioteca Pandas.

O objetivo é praticar a importação, exploração, filtragem, agrupamento e análise de dados.

## Perguntas de negócio

A análise procura responder às seguintes perguntas:

- Qual é o imóvel mais caro?
- Qual é o imóvel mais barato?
- Qual é o valor médio dos imóveis por tipo?
- Quantos imóveis possuem 3 ou mais quartos?
- Quantos imóveis possuem 3 ou mais quartos e 2 ou mais vagas de garagem?
- Quais tipos de imóveis possuem os maiores valores médios?
- Quais bairros possuem mais imóveis disponíveis?

## Tecnologias utilizadas

## Arquivos do projeto

- `aluguel.csv`: base de dados utilizada na análise.
- `analise_imoveis.ipynb`: notebook que será utilizado para desenvolver a análise.

## Conhecimentos praticados

Durante o desenvolvimento deste projeto, serão praticados:

- Leitura de arquivos CSV
- Exploração de DataFrames
- Seleção de colunas
- Filtros com uma ou mais condições
- Ordenação de dados
- Agrupamento com `groupby()`
- Cálculo de médias e contagens
- Identificação de valores máximos e mínimos
- Apresentação de resultados em tabelas
