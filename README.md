
# 🏠 Mini Projeto: Análise de Imóveis com Pandas



## 📌 Sobre o projeto

Este projeto apresenta uma análise exploratória de uma base de dados de imóveis para aluguel.

O trabalho foi desenvolvido com o objetivo de praticar conceitos fundamentais de análise de dados utilizando **Python** e a biblioteca **Pandas**.

Durante a análise, foram aplicadas técnicas de importação, exploração, filtragem, agrupamento, ordenação e apresentação de dados.

---

## 🎯 Objetivo

Explorar uma base de imóveis e transformar os dados disponíveis em respostas para perguntas de negócio.

A proposta é simular algumas das atividades realizadas por um analista de dados ao investigar uma base e comunicar suas principais descobertas.

---

## ❓ Perguntas de negócio

A análise foi desenvolvida para responder às seguintes perguntas:

1. **Quantos imóveis existem na base de dados?**
2. **Quantos tipos diferentes de imóveis existem?**
3. **Qual bairro possui a maior quantidade de imóveis cadastrados?**
4. **Qual é o valor médio dos imóveis?**
5. **Qual é o imóvel mais caro e qual é o mais barato?**

## 🧠 Conhecimentos praticados

Durante o desenvolvimento deste projeto, foram utilizados os seguintes recursos do Pandas:

- `read_csv()` para importar a base de dados
- `head()` para visualizar os primeiros registros
- `info()` para verificar a estrutura do DataFrame
- `shape` para consultar a quantidade de linhas e colunas
- `value_counts()` para calcular frequências
- `groupby()` para agrupar informações
- `mean()` para calcular médias
- `sort_values()` para ordenar os dados
- `head(1)` para selecionar o primeiro resultado
- `reset_index()` para transformar resultados em tabelas
